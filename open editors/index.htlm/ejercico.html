HTML

HyperText Markup Languaje o como lo llamamos HTML es un lenguaje de marcado, el cual por medio de etiquetas describimos el contenido de lo que será una página web.

  • HTML no se ocupa de la presentacion, de eso se encarga CSS.
  • HTML no se ocupa de la funcionalidad, para eso se necesita JavaScript.
  • HTML no se de describir el contenido de una pagina web.

Los elementos de HTML se componen por las siguientes estructuras:

Estructura basica de un documento HTML.

Existe etiquetas para representar diferentes elementos visuales.

  • Textos.
  • Listas.
  • Enlaces.
  • Imagenes.
  • Formularios.

Etiquetas de texto.

  • Etiquetas de encabezados: <h1></h1>, <h2></h2>, <h3></h3>, <h4></h4>, <h5></h5>, <h6></h6>
  • Etiquetas de parrafo: <p></p>
  • Etiqueta de negrita: <strong></strong>
  • Etiqueta cursiva: <i></i>
