HTML

HyperText Markup Languaje (HTML) es el codigo que se utiliza para estucturar y desplegar una pagina web y sus contenidos

Ejemplo

h1:

sirve para el cierre de apertura

title:

sirve para darle el titulo a la pagina web

body:

inicia la parte del cuerpo del contenido de la pagina web

p:

sirve para crear los parrafos

Hola mundo:
